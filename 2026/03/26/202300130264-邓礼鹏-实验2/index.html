云计算技术 课程实验报告
 
学号：202300130264 姓名：邓礼鹏 班级：2班
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2026/3/20
实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。
具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 
联网的计算机一台

软件环境：
Windows or Linux

实验步骤与内容：
一、本地搭建
1.准备工作：
注册登录GitHub账号：https://github.com/hellotheworld82
安装git：https://git-scm.com/
安装Node.js：https://nodejs.org/，选择 LTS 版本，默认安装
记住用户名，后面要用
2. 搭建 Hexo 博客
打开终端，逐行执行：
npm install -g hexo-cli

hexo init myblog
cd myblog

npm install

hexo s

执行后打开浏览器访问：http://localhost:4000

看到博客页面

则环境搭建成功
二、把博客发布到互联网

  1. 创建 GitHub 仓库
    仓库名：hellotheworld82.github.io

  2. 配置 Hexo 部署
    打开博客根目录（C:\Users\li\myblog）的 _config.yml 文件，拉到最底部修改：
    yaml
    deploy:
    type: git
    repo: https://github.com/hellotheworld82/hellotheworld82.github.io.git
    branch: main

  3. 安装部署插件
    npm install hexo-deployer-git –save

  4. 一键发布上线
    hexo clean
    hexo g
    hexo d
    发布完成后，访问：https://hellotheworld82.github.io

三、把实验报告发布到博客

  1. 新建文章
    在博客目录执行命令：
    hexo new “202300130264-邓礼鹏-实验2”
  2. 编辑文章
    打开目录：source/_posts/202300130264-邓礼鹏-实验2.md
    把本次实验报告粘贴进去，保存。
  3. 重新发布
    hexo g
    hexo d
    刷新博客网站文章出现在首页

结论分析与体会：
本次实验成功搭建了基于 Hexo+GitHub Pages 的个人博客，完成了从环境配置到公网部署的全流程，并将实验报告作为博文发布。通过实践，我掌握了静态网站生成与部署的核心方法，解决了网络卡顿、配置错误等问题，体会到静态博客低成本、高可靠的优势，也提升了技术文档编写与问题排查能力。
